Here is the rewritten text about the Swift programming language, condensed to a similar volume:
SWIFT - A Programming Language for iPhone App Development
Swift is a modern, high-performance language developed by Apple for building iOS, macOS, watchOS, 
and tvOS apps. It's designed to give developers the ability to create powerful, modern apps with a 
clean and easy-to-read syntax.
                
Xcode: The Most Popular Development Environment
Xcode is the most popular integrated development environment (IDE) for Swift, providing a comprehensive 
set of tools for coding, debugging, and testing apps. Xcode offers two programming languages: 
Swift and Objective-C (which is now considered legacy).
Key Features of Swift
Swift is known for its simplicity, readability, and ease of use. It's a statically typed 
language, which means it checks for errors at compile time, preventing runtime errors. 
Swift is also designed to 
work seamlessly with Apple's Cocoa and Cocoa Touch frameworks, making it an ideal choice for 
iOS app development.
Here's a simple "Hello, World!" example in Swift:


import UIKit
                
class ViewController: UIViewController {
override func viewDidLoad() {
    super.viewDidLoad()
    print("Hello, World!")
    }
}


Overall, Swift is a powerful and intuitive language that's perfect 
for building high-quality, engaging apps for Apple devices.
